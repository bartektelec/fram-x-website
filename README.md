# fram-x-website
project of a new website for company Fram X

## SCSS files
uncompiled scss files can be found in `scss` folder
every section of the website has its own scss file
`_comp.scss`has styles for _components/modules_
`_type.scss`has styles for _typography_
`ie9.scss`has IE9 specific styles


## source files
.ai files for adobe illustrator are to be found in `img/source`

## images

`logo`folder contains logos of different partner companies

`pictures`folder contains pictures of Fram X team

`svg` folder contains other illustrations used on the website
